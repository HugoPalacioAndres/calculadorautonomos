2. Datos que recopilamos

Este sitio web no recopila datos personales identificables de forma directa. Sin embargo, si el usuario acepta las cookies analíticas, se recopilan los siguientes datos de forma anónima mediante Google Analytics:

  • Páginas visitadas y tiempo de permanencia
  • País y ciudad de origen (aproximado)
  • Dispositivo y navegador utilizado
  • Fuente de tráfico (buscador, directo, redes sociales)

3. Finalidad del tratamiento

Los datos analíticos se utilizan exclusivamente para:

  • Mejorar los contenidos y herramientas del sitio web
  • Analizar el comportamiento de los usuarios de forma agregada y anónima
  • Optimizar la experiencia de navegación

4. Base legal

El tratamiento de datos analíticos se basa en el consentimiento expreso del usuario mediante el banner de cookies. El usuario puede retirar su consentimiento en cualquier momento desde la política de cookies.

5. Transferencias internacionales

Google Analytics puede transferir datos a servidores ubicados en Estados Unidos. Google LLC está adherida al marco EU-US Data Privacy Framework, lo que garantiza un nivel de protección adecuado.

6. Derechos del usuario

El usuario tiene derecho a acceder, rectificar, suprimir, oponerse y limitar el tratamiento de sus datos. Para ejercer estos derechos puede contactar en: info@calculadorautonomos.es

7. Conservación de datos

Los datos analíticos se conservan durante un máximo de 14 meses según la configuración predeterminada de Google Analytics 4.
